# УСТАНОВКА И ЗАПУСК Parser Bot 🚀

## 📋 Что вам нужно

1. **Python 3.8+** - https://www.python.org/downloads/
2. **Токен Telegram бота** - от @BotFather в Telegram  
3. **Git** (опционально) - для клонирования репо

## 🔑 Получение Telegram токена

1. Откройте Telegram и напишите **@BotFather**
2. Отправьте команду: `/newbot`
3. Придумайте имя для бота
4. Придумайте username (например, `my_parser_bot`)
5. Получите токен вида: `123456789:ABCdefGHIjklmnoPQRstuvWXYZ`
6. **Сохраните этот токен** - вам понадобится!

## 🚀 Быстрый запуск (Windows)

### Шаг 1: Подготовка
```bash
# Откройте командную строку (cmd.exe) или PowerShell
# Перейдите в папку проекта
cd C:\Users\Askhat\OneDrive\Рабочий стол\проекты\parser
```

### Шаг 2: Создание .env файла
```bash
# Скопируйте пример файла
copy .env.example .env

# Откройте .env в блокноте
notepad .env
```

**Отредактируйте файл:**
```
TELEGRAM_BOT_TOKEN=СЮДА_ВСТАВЬТЕ_ВАШ_ТОКЕН_БОТА
DATABASE_URL=sqlite:///./parser_data.db
LOG_LEVEL=INFO
API_TIMEOUT=30
MAX_WORKERS=5
```

### Шаг 3: Установка зависимостей
```bash
# Создание виртуального окружения
python -m venv venv

# Активирование окружения
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt
```

### Шаг 4: Запуск бота
```bash
python bot.py
```

Вы должны увидеть:
```
✅ Бот запущен! Нажмите Ctrl+C для остановки.
```

### Шаг 5: Тестирование
1. Откройте Telegram
2. Найдите вашего бота (по username)
3. Отправьте команду `/start`
4. Попробуйте выполнить поиск

## 🎯 Первый поиск - Шаг за шагом

### Опция 1: Поиск на 2ГИС

1. Нажмите "🔍 **Новый поиск**"
2. Выберите "**2ГИС**"
3. Выберите город: "**Алматы**" (или другой)
4. Введите запрос: "**кафе**" или "**парикмахерская**"
5. Ждите результатов (10-30 секунд)
6. Нажмите "✅ **Готово**" или "**📊 Просмотреть**"

### Опция 2: Поиск на Kaspi

1. Нажмите "🔍 **Новый поиск**"
2. Выберите "**Kaspi**"
3. Введите запрос: "**электроника**" или "**одежда**"
4. Ждите результатов
5. Просмотрите результаты

### Опция 3: Поиск в оба источника

1. Нажмите "🔍 **Новый поиск**"
2. Выберите "**Оба источника**"
3. Выберите город
4. Введите запрос
5. Дождитесь результатов

## 🐛 Решение проблем

### Проблема: ModuleNotFoundError
**Решение:**
```bash
# Убедитесь что окружение активировано
venv\Scripts\activate

# Переустановите зависимости
pip install -r requirements.txt --force-reinstall
```

### Проблема: Бот не отвечает
**Решение:**
1. Проверьте токен в файле `.env`
2. Убедитесь бот запущен (ищите строку "✅ Бот запущен!")
3. Проверьте интернет соединение
4. Перезапустите бота (Ctrl+C и `python bot.py`)

### Проблема: Ошибка при парсинге
**Решение:**
1. Проверьте файл логов: `logs/parser.log`
2. Убедитесь что сайты 2GIS и Kaspi доступны
3. Попробуйте включить VPN
4. Проверьте параметры поиска (используйте более конкретный запрос)

### Проблема: "TELEGRAM_BOT_TOKEN не установлен"
**Решение:**
1. Откройте файл `.env`
2. Убедитесь что токен вставлен правильно
3. Боект не должно быть пробелов вокруг `=`
4. Сохраните файл
5. Перезапустите бота

## 🚀 Развёртывание на сервер

### Используя Docker (рекомендуется)
```bash
# Создайте .env с вашим токеном
cp .env.example .env
# Отредактируйте .env

# Запустите
docker-compose up -d

# Проверьте логи
docker-compose logs -f parser-bot
```

### На Linux VPS
```bash
ssh user@your-server.com

# Клонируйте проект
git clone <your-repo>
cd parser

# Установите
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Создайте .env
cp .env.example .env
nano .env  # отредактируйте

# Запустите в фоне
tmux new-session -d -s parser python bot.py

# Проверьте
tmux attach-session -t parser
```
